# formacion-web-dev-basic-loops-second-task

Este repositorio ha sido adaptado para contener 10 ejercicios centrados en metodos de arrays de JavaScript:

Cada ejercicio está implementado en `src/index.js` con nombres y enunciados.

Los tests están en `src/test.js` y generan pruebas que verifican la forma del resultado
(tipo/estructura) con entradas canónicas. El sistema de ejecución y la UI en `index.html` se han respetado.

Cómo usar:

1. Abre `index.html` en tu navegador.
2. Haz clic en "Run tests" para ejecutar las pruebas y ver los resultados en pantalla.

Notas:
- Los tests actuales comprueban la forma del resultado (tipo/estructura). Puedes ampliar los tests
	para comprobar la lógica detallada de cada ejercicio si lo deseas.
